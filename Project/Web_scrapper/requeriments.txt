1. You need to install the same chromedriver version as you Google Chrome browser. 
you can install it in the following url: https://chromedriver.chromium.org/downloads

2. Install selenium with the following command: pip install selenium

3. Download the files in the same folder

4. In the scrapper.py file add an username and a password so the code can search the account and download the info with it.

5. Run the main.py since it calls of the necessary functions to perform the scrapper and get a json file
